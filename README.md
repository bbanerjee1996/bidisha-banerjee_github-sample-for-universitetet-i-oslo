# Agrivoltaics Techno-Economic Assessment Model

This repository contains a techno-economic financial model developed for evaluating the feasibility of a grid-connected agrivoltaics system in India.

## Overview
The model evaluates:
- Capital expenditure (CAPEX)
- Debt-equity financing structure
- Annual energy generation
- O&M escalation
- Depreciation
- Working capital
- Net Present Value (NPV)
- Internal Rate of Return (IRR)
- Payback period
- Agricultural co-benefits

## Project Details
- Plant Capacity: 0.6 MW
- Technology: Grid-connected Agrivoltaics
- Project Lifetime: 25 years

## Tools Used
- Microsoft Excel
- Financial modelling
- Renewable energy system analysis

## Research Context
The model was developed as part of techno-economic analysis work related to agrivoltaics and renewable energy deployment in India. It evaluates long-term system feasibility under varying operational and financial conditions.
